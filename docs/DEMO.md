# Two-minute local demo

1. Start `make api` and `make web` in separate terminals.
2. Upload a small `.xlsx` sales sheet and select its sheet and target cell.
3. Enter “Calculate total sales for this row” and choose **Forge formula**.
4. Point out the formula, validation state, preview value, local model label, and latency.
5. Switch to Repair and show the explicit current-formula field.
6. Demonstrate an invalid workbook or unsupported formula state, then note that download requires explicit confirmation.
